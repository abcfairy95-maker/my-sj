# 楚雄师范学院官网信息

以下信息整理自楚雄师范学院官方网站（www.cxtc.edu.cn）。

## 学校简介

楚雄师范学院位于云南省楚雄彝族自治州府楚雄市，是一所具有90年师范教育积淀，教师教育特色鲜明，多学科协调发展的省属本科高校。

历史沿革：学校溯源于1936年建校的省立镇南师范学校。1953年更名为楚雄师范学校，1984年更名为楚雄民族师范学校。1978年创建楚雄师范专科学校，1992年更名为楚雄师范高等专科学校。1999年楚雄民族师范学校并入楚雄师范高等专科学校。2001年5月经教育部批准升格为本科层次的楚雄师范学院。

学校现有12个教学学院，49个全日制本科专业。全日制本科生13651人。专任教师818人，其中教授91人、副教授216人、博士238人。学校占地面积415788平方米，纸质图书137万册。

## 院系设置与专业

楚雄师范学院设有12个教学学院，各学院及其开设的本科专业如下。教育学院开设的专业有学前教育、小学教育、教育技术学、应用心理学。

**马克思主义学院**：思想政治教育

**教育学院**：学前教育、小学教育、教育技术学、应用心理学

**人文学院**：汉语言文学、汉语国际教育、秘书学、历史学

**外国语学院**：英语、翻译、商务英语

**计算机与智能技术学院**：计算机科学与技术、网络工程、数据科学与大数据技术、人工智能、教育技术学（与教育学院合办）

**数学与统计学院**：数学与应用数学、统计学、应用统计学、信息与计算科学

**物理与电气能源工程学院**：物理学、电气工程及其自动化、新能源科学与工程、电子信息科学与技术

**化学与地理科学学院**：化学、应用化学、地理科学、自然地理与资源环境、地理信息科学

**管理与经济学院**：会计学、财务管理、工商管理、旅游管理、市场营销、物流管理

**体育与健康学院**：体育教育、社会体育指导与管理、运动康复

**美术与艺术设计学院**：美术学、视觉传达设计、环境设计、产品设计、服装与服饰设计、数字媒体艺术、书法学

**继续教育学院**：成人高等教育、函授教育、非学历培训

注：以上专业信息综合整理自学校官网。学校现有49个全日制本科专业，其中国家级特色专业1个（小学教育），国家级一流专业建设点2个，省级一流专业建设点11个。

## 学校标识

楚雄师范学院的官方校训是"厚德、博学、求是、创新"，请以此为准。校训不是"勤奋、严谨、务实"，那个是校风。
校风：团结、勤奋、严谨、务实。
办学特色：以师范教育为主，多学科协调发展。
