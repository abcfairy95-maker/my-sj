## 问题与解决（Debug 案例）

### Bug 1：split_text 死循环
**症状**：文档切片时程序卡死  
**原因**：最后一段切片到达文档末尾后，`start = end - overlap` 退回之前的起始位置，导致无限循环  
**修复**：当 `end == text_len` 时直接 break  
```python
if end == text_len:
    break
```

### Bug 2：Qwen Embedding 批量大小超限
**症状**：构建索引时报错 `batch size is invalid, it should not be larger than 10`  
**原因**：Qwen API 限制单次请求最多 10 条文本，但代码设的是 100  
**修复**：将 `EMBEDDING_BATCH_SIZE` 改为 10

### Bug 3：Gradio 6.x API 不兼容
**症状**：`theme`、`retry_btn`、`placeholder` 等参数报错  
**原因**：代码基于 Gradio 4/5 版本编写，Gradio 6 重构了 API  
**修复**：将 `theme` 和 `css` 移到 `launch()` 方法，移除不兼容的按钮参数

### Bug 4：文档来源未显示
**症状**：回答没有"参考来源"显示  
**原因**：`build_index.py` 写入时未传入 `metadatas`  
**修复**：让 `load_and_split()` 返回 `(chunks, sources)`，重建索引时携带来源信息
