# 楚雄师范学院学生社团与组织指南

## 第一章 学生组织

### 1.1 学生会
- **校学生会**：最高学生自治组织，设以下部门：
  - 主席团：统筹学生会全面工作
  - 办公室：文档管理、会议记录、物资管理
  - 学习部：学风建设、学习经验分享、学术讲座
  - 文艺部：策划组织文艺晚会、歌手大赛
  - 体育部：各类体育赛事组织
  - 外联部：校际交流、赞助合作
  - 宣传部：活动宣传、新媒体运营
  - 生活部：宿舍卫生检查、权益维护

- **院系学生会**：各二级学院分别设立，负责本学院学生工作
- **招新时间**：每年 9-10 月新生入学后统一招新
- **加入方式**：填写报名表 → 面试 → 试用期 1 个月 → 正式录用
- **学生干部待遇**：
  - 综合测评加分（校级干部加 5 分，院级加 3 分）
  - 评优评先优先
  - 可参评优秀学生干部奖学金
  - 推荐参加省级以上培训交流活动

### 1.2 团委组织
- **校团委**：负责全校团员管理和思想政治教育工作
- **团总支**：各学院设立团总支
- **团支部**：以班级为单位设立，每班一个团支部
- **入团流程**：提交申请书 → 团支部考察（3 个月）→ 团课学习（8 学时）→ 宣誓入团
- **团员义务**：按时缴纳团费（2 元/月）、参加团日活动
- **推优入党**：优秀团员可经团支部推荐发展为预备党员
  - 条件：已递交入党申请书满 1 年、团龄 1 年以上、品学兼优
  - 流程：团支部推荐 → 学院团委审核 → 校团委审定 → 党支部发展

### 1.3 学生社团联合会
- **办公地点**：学生活动中心 2 楼 204 室
- **值班时间**：周一至周五 16:00-18:00
- **主要职责**：
  - 学生社团的注册、注销和年审
  - 社团活动的审批和备案
  - 社团经费的监督和审计
  - 社团干部的培训和管理
  - 社团评优的组织工作

### 1.4 志愿者组织
- **青年志愿者协会**：校级志愿者组织
- **各学院志愿服务队**：每个学院有独立志愿服务队
- **注册志愿者**：在"志愿中国"网站注册，可记录志愿服务时长
- **服务内容**：社区服务、支教、敬老院慰问、环境保护、大型活动志愿服务

### 2.2 如何加入社团
- **百团大战**：每年 9 月开学第二周，在图书馆前广场举行社团联合招新
- **平时加入**：可直接联系社长申请加入
- **会费**：多数社团免会费，部分社团收取 10-50 元/年用于活动材料购买
- **加入限制**：建议加入不超过 3 个社团，以免影响学习

### 2.3 社团活动学分认定
- 参与社团活动满一学年并通过考核，可获得 **1 个选修学分**
- 认定条件：出勤率 ≥ 70% + 完成社团布置的考核任务
- 每学期末由社团联合会统一认证上报

## 第四章 综合测评

### 4.1 测评构成
学生综合测评满分 100 分，构成如下：
- **学习成绩**（60%）：期末成绩加权平均
- **思想品德**（15%）：政治表现、遵纪守法、道德品质
- **文体活动**（15%）：参与社团、比赛、晚会等
- **社会实践**（10%）：志愿服务、实习实践

### 4.2 加分细则
| 加分项 | 分值 | 认定部门 |
|--------|------|---------|
| 担任学生干部（校级） | 5 分 | 团委 |
| 担任学生干部（院级） | 3 分 | 学院 |
| 参与社团活动（每次） | 0.5 分（上限 5 分） | 社团联合会 |
| 学科竞赛获奖（校级） | 1-3 分 | 教务处 |
| 学科竞赛获奖（省级） | 3-8 分 | 教务处 |
| 学科竞赛获奖（国家级） | 8-15 分 | 教务处 |
| 志愿服务（每小时） | 0.1 分（上限 5 分） | 青志协 |
| 发表论文（核心期刊） | 10 分 | 科研处 |
| 发表论文（普通期刊） | 5 分 | 科研处 |
| 英语四六级通过 | 2 分 | 学院 |
| 计算机等级考试通过 | 1 分 | 学院 |

### 4.3 测评用途
综合测评成绩直接与以下事项挂钩：
- 奖学金评定（国家奖学金、校级奖学金）
- 评优评先（三好学生、优秀学生干部）
- 推优入党
- 保研资格
- 毕业推荐

综合测评每学年评定一次，由辅导员和班委组成的测评小组负责。
