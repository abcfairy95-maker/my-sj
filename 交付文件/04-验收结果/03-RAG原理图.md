## RAG 原理图

```
用户提问："图书馆周末几点开门？"
         │
         ▼
┌─────────────────────────┐
│  ① 检索 (Retrieval)     │
│  问题 → Embedding → 向量 │
│  在 ChromaDB 中检索 Top3 │
│  返回相关文档片段        │
└─────────┬───────────────┘
          │
          ▼
┌─────────────────────────┐
│  ② 增强 (Augmentation)  │
│  检索结果 + 用户问题     │
│  拼接为完整 Prompt       │
│  格式：[参考文档] + [问题]│
└─────────┬───────────────┘
          │
          ▼
┌─────────────────────────┐
│  ③ 生成 (Generation)    │
│  DeepSeek 根据文档回答    │
│  返回准确、有据可依的答案 │
└─────────────────────────┘
          │
          ▼
"图书馆周末开放时间为 09:00-21:00"
📚 参考来源：图书馆使用指南.md
```
